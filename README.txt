#GPSMAPAPP

La utilidad de la aplicacion es localizar el dispocitivo en un mapa si se conceden los permidos necesarios

##Funcionalidades

-Mostrar la ubicaion actual del dispositivo en tiempo real

-Incluye una brujula

-Incluye direccion de moviento y la muestra en la pantalla

##Como ejecutarla
1. Clona el repositorio: `git clone https://github.com/Solar707/GpsMappApp.git
2. Abre el proyecto en Android Studio.
3. Añade tu API Key de Google Maps en `res/values/google_maps_api.xml".
4. Ejecuta la aplicación en un dispositivo o emulador Android.
